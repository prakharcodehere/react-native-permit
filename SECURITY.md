# Security Policy

## Supported versions

Security fixes are expected to target the latest published version.

| Version | Supported |
| --- | --- |
| `0.1.x` | Yes |

## Reporting a vulnerability

Please report security issues privately through the GitHub repository once it is
public:

`https://github.com/prakharcodehere/react-native-permit/security/advisories`

Do not open public issues for vulnerabilities.

## Scope

`react-native-permit` does not ship native code and does not request permissions
directly from iOS or Android. Native behavior is delegated to the
`react-native-permissions` peer dependency.
